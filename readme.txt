Auto-resize your post thumbnails when the config are changed, and fix the thumbnail names.

== Description ==
This plugin do two simple things.
* Auto-resize the thumbnail.
* Fix the thumbnail names. change imgname_100x100.jpb by imagename_thumb.jpg

== Installation ==
1. Download the plugin Zip archive.
2. Upload "photoblog" dir to your /wp-content/plugins/ directory
3. Install a Theme
4. Activate the plugin through the 'Plugins' menu in WordPress
9. Enjoy.
